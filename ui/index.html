🔧 Setup Guide

Quickstart

Step 1  pip install -r requirements.txt
Step 2  Enter all API keys in the ⚙️ Settings tab above — no .env needed
Step 3  python setup_trunk.py → syncs Vobiz to LiveKit (one-time)
Step 4  python -c "from db import init_db; init_db()"
Step 5  Terminal 1: python agent.py start
Step 6  Terminal 2: uvicorn server:app --host 0.0.0.0 --port 8000 --reload
Step 7  Open http://localhost:8000 and use Single Call to test 🎉
